# dadnowcast

This repo implements an R package named `dadnowcast` that aims to nowcast DAD hospital admissions given other data sources (like RVDSS, PTSOS, CNISP, ...). 

See [`vignette("Example_Usage", package = "dadnowcast")`](https://github.com/DASL-Lab/dadnowcast/blob/main/vignettes/Example_Usage.Rmd) for package details.
